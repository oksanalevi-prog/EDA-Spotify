# 🎵 Spotify — Exploratory Data Analysis (EDA)

## Descrizione del progetto
Analisi esplorativa di un dataset Spotify composto da **8.582 tracce** e **18 variabili**.
L'obiettivo è comprendere la struttura dei dati, identificare pattern e relazioni 
tra le variabili musicali.

## Obiettivi dell'analisi
- Esplorare la struttura e la distribuzione delle variabili
- Individuare valori mancanti e outlier
- Analizzare correlazioni tra variabili
- Formulare ipotesi su popolarità, artisti e generi musicali

## Principali risultati
- 🎸 Il genere **country, pop** domina per numero di tracce e popolarità
- 👩‍🎤 **Taylor Swift** è l'artista con la maggiore produzione musicale nel dataset
- 📅 Le tracce degli **anni '80** risultano tra le più popolari su Spotify oggi
- 🚀 Gli artisti emergenti con un solo album mostrano spesso popolarità media > 60
